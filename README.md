# APN Manager (Custom Edition)

Это персональный инструмент для управления профилями APN в модемах Huawei (HiLink), разработанный для автоматизации настройки специфических параметров связи.

> **Важно:** Данное приложение настроено под личные задачи автора. Если вы планируете использовать его для своих целей, вам может потребоваться изменить исходный код.

## Особенности
* **Быстрая настройка:** Кнопка для мгновенного добавления предустановленного профиля (Sputnikovaya Svyaz).
* **Управление APN:** Возможность переключения и создания профилей без входа в веб-интерфейс модема.
* **Автоматизация:** Встроенная проверка связи (Ping) после смены настроек.

## Как адаптировать под себя

Если вам нужно изменить предустановленное имя профиля или адрес APN, внесите правки в файл `apn_dodi.py` перед сборкой:

1. Откройте `apn_dodi.py`.
2. Найдите функцию `quick_add_sputnik`.
3. Измените значения переменных:
   ```python
   name = "Ваше Название"
   apn = "ваш.apn.адрес"
   ```
4. Сохраните файл и залейте на GitHub — автоматика сама соберет новые версии под Windows и Mac.

## Инструкция по запуску

### Windows
1. Скачайте `APN_Manager.exe` из раздела Artifacts.
2. При первом запуске может потребоваться разрешение от SmartScreen (нажмите "Подробнее" -> "Выполнить в любом случае").

### macOS
1. Скачайте архив `APN_Manager_MacOS`.
2. Распакуйте и перетащите `APN_Manager.app` в папку "Программы".
3. **Для запуска:** Нажмите на файл **правой кнопкой мыши** и выберите **Открыть**. Это необходимо, так как приложение не имеет подписи Apple.
