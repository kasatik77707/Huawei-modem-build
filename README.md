# APN Manager (Huawei E3372)

**APN Manager** — это компактное и удобное приложение для управления профилями APN в USB-модемах Huawei (в частности, модели E3372). Приложение позволяет быстро переключаться между предустановленными тарифами, создавать новые конфигурации и отслеживать состояние подключения в реальном времени.

## Основные возможности

* **Переключение APN:** Мгновенная смена активного профиля без необходимости входа в веб-интерфейс модема.
* **Создание профилей:** Упрощенное добавление новых APN (защищено паролем администратора).
* **Статус сети:** Встроенный индикатор Ping для проверки фактического наличия интернета.
* **Журнал событий:** Детальное логирование всех операций и ответов оборудования.
* **Кроссплатформенность:** Поддержка Windows (.exe) и macOS (.app).

## Системные требования

* **Модем:** Huawei E3372 (или аналоги с поддержкой HiLink API).
* **Адрес модема:** По умолчанию `192.168.8.1`.
* **Логин/Пароль модема:** `admin / admin`.

## Установка и запуск

### Для пользователей Windows
1. Скачайте архив из раздела **Releases** или **Artifacts**.
2. Распакуйте и запустите файл `APN_Manager.exe`.
3. Если антивирус или брандмауэр блокирует программу (из-за отсутствия цифровой подписи), добавьте её в исключения или разрешите запуск.

### Для пользователей macOS
1. Скачайте и распакуйте архив `APN_Manager_MacOS`.
2. Перетащите файл `APN_Manager.app` в папку **Программы**.
3. **Первый запуск:** Нажмите на приложение правой кнопкой мыши и выберите **Открыть**, затем подтвердите выбор. Это необходимо для обхода системной защиты (Gatekeeper).

## Безопасность

* Доступ к созданию новых профилей по умолчанию защищен паролем: `1234`.
* Приложение работает только локально и не передает ваши данные на внешние серверы.

## Разработка

Проект написан на **Python** с использованием следующих библиотек:
* `customtkinter` — современный графический интерфейс.
* `huawei-lte-api` — библиотека для взаимодействия с модемом.

### Сборка из исходного кода
Если вы хотите собрать приложение самостоятельно, используйте **PyInstaller**:

```bash
# Для Windows
pyinstaller --noconsole --onefile --collect-all customtkinter --name "APN_Manager" apn_change.py

# Для macOS
python3 -m PyInstaller --noconsole --onefile --collect-all customtkinter --name "APN_Manager" apn_change.py
```

## Автоматизация сборки
В репозитории настроен **GitHub Actions**. При каждом обновлении кода в ветке `main`, облачные серверы GitHub автоматически собирают свежую версию приложения для macOS.
